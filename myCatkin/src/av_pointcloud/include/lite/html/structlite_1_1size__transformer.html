Detailed Description

template<typename transform_type_, typename input_size_type_>
struct lite::size_transformer< transform_type_, input_size_type_ >

This can be used to apply a transform object of type transform_type_ to a size object of type input_size_type_.

See Defining New Transforms for more information and examples.

Apply the transform to the size object org and store the result in res.

Remarks:
This overload of the transform() function is optional. Not all transforms provide this overload. It is recommended that you provide this overloaded version if an actual transform object does not carry any information required for applying the transform (e.g. like lite::transpose).
